# 《定格：向光而行》

## 作品简介

本组作品围绕“寻帧·知夏”主题展开创作，以华中科技大学校园为背景，通过花海、花坛与计算机学院建筑三个场景，记录属于计算机学院学子的夏日成长轨迹。

在计算机科学中，“帧”是记录变化的基本单位；在摄影中，“帧”则是定格时光的载体。本组作品尝试以三个关键帧，呈现青春、成长与求索的过程，将校园生活中的瞬间凝聚为关于理想与未来的叙事。

---

## 第一帧：《星途》

照片拍摄于校园花海。

盛放的花朵铺展开来，镜头聚焦于中心校牌。六月的校园正值毕业季，花海既是夏天到来的讯号，也象征着青年学子即将踏上新的征程。

对于学生而言，大学时光是梦想萌芽与积蓄力量的阶段。花海中的每一朵花都在奔赴属于自己的花期，而每一位即将离开校园的毕业生，也正在走向更广阔的天地。

这一帧记录的不仅是夏日的色彩，更是青春对于未来的期待与想象。

---

## 第二帧：《生长》

照片聚焦于教学楼花坛中的一朵小花。

与大片花海相比，它并不起眼，却在镜头中独自向上舒展。仰拍视角让花朵与天空相接，也赋予了画面一种突破与成长的力量感。

成长往往不是轰轰烈烈的绽放，而是在平凡的日子里不断向上生长。正如华科莘莘学子，在在求学生涯里逐渐成长，在一次次尝试与坚持中完成自我突破。

这一帧记录的是个体的成长，也是青春最真实的模样。

---

## 第三帧：《求索》

照片拍摄于计算机学院大楼。

建筑采用仰拍构图，前景的绿色植物与建筑线条共同向上延伸。理性的建筑结构与自然生命力在同一画面中相互映衬，构成科技与人文交融的校园景观。

作为计算机学院的学生，我们每天与代码、算法和数据打交道。理性的逻辑背后，始终蕴含着对未知世界的探索热情。计算机科学追求严谨与理性，而青年成长同样需要坚定的方向与持续的探索。

学院大楼不仅是知识汇聚的空间，也承载着无数关于创新、奋斗与梦想的故事。

这一帧记录的是专业精神，也是青年求索未来的起点。

---

## 创作思考

从花海到单花，从自然景观到学院建筑，三张照片分别对应未来、成长与求索。

毕业花海象征远方与梦想；
单个花朵象征个体与成长；
学院建筑象征知识与力量。

三者共同记录华科学子、计科人的成长轨迹：在校园中扎根，在奋斗中成长，在理性的知识世界中不断探索，最终奔赴属于自己的星辰大海。

夏天不仅是季节的更替，更是成长与出发的时刻。希望通过这三个关键帧，记录属于华中科技大学计算机学院学子的青春印记，也记录那些在时光流转中依然闪耀的理想与热爱。
